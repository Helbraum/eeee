These are the steps to push a github repository

First lets discuss what is git
Git is a free and open source distributed control system design to handle
everything from small to very large projects with speed and efficiency

Here are the steps to push your code to your own repository

    1. Make sure you're in the right directory:

        pwd <- print working directory

    2. Make sure you configure your credentials

        git config --global user.email[user github email]

    3. Add your remote repository in your local machine

    4. Make sure you're in the correct branch.

        git branch -M main 

    5. check your file using git status

        git status

    6. Add all your code to the add phase

        git add -A 

    7. Then commit your code to the repository

        git commit -m "Commit code"

    8. Push your code to the repository

        git push -u (localCompuuter)
        (currentBranch)
